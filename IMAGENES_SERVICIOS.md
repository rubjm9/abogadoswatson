## Notas Técnicas

- Todas las imágenes se muestran con `opacity-40` para mantener la legibilidad del texto
- Se aplica una capa oscura (`bg-slate-900/80` o similar) sobre la imagen
- Las imágenes usan `object-cover` para cubrir todo el área del hero
- Si prefieres usar imágenes locales, descárgalas desde Unsplash y guárdalas en `/public/images/`

## Cómo Descargar Imágenes Locales

1. Visita cada URL de Unsplash en el navegador
2. Haz clic en el botón de descarga
3. Guarda la imagen en `/public/images/` con el nombre sugerido
4. Actualiza las páginas para usar la ruta local en lugar de la URL de Unsplash
